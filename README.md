# Mathle Starter

수학적으로 성립하는 수식을 입력하고, Wordle 방식 피드백으로 정답 수식을 맞히는 웹 게임 스타터 프로젝트입니다.

## 실행
```bash
npm install
npm run dev
```

## 현재 포함 기능
- 퍼즐 API
- 결과 저장 API
- 리더보드 API
- 프론트 게임 UI
- 수식 검증 / 평가 (mathjs)
- 변수 1개 예시
- 공유 문자열 복사

## 현재 제약
- 결과 저장은 `data/results.json` 파일 기반이라 로컬/단일 서버 데모용입니다.
- 서버리스 배포에서는 영속 저장이 되지 않으므로 추후 DB로 바꾸는 것을 권장합니다.
